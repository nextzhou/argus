# Support

## Where to Ask for Help

Use GitHub Issues for:

- Usage questions
- Installation problems
- Troubleshooting hook integrations
- Bug reports and unexpected behavior
- Feature requests

## Before Opening an Issue

- Read [README.md](README.md)
- Check [docs/reference.md](docs/reference.md) for CLI and authoring details
- Check [docs/technical-overview.md](docs/technical-overview.md) if your question is about internal behavior or architecture
- Search existing issues to avoid duplicates

## What to Include

Please include as much of the following as you can:

- What you were trying to do
- What you expected to happen
- What actually happened
- Relevant command output or logs
- Your operating system, shell, and agent integration details
- Reproduction steps or a minimal sample repository when possible
